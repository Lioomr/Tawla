# SKILL: Frontend UI Direction

## Purpose

Keep Tawlax frontend modern, simple, fast, and visually intentional.

---

## Mandatory Direction

Frontend must be:

* Modern
* Minimal
* Fast
* Creative
* Operationally clear

---

## Rules

* Do not ship generic template layouts
* Do not hide core actions behind extra taps
* Use clear hierarchy before decorative elements
* Keep customer flows calm and fast
* Keep staff dashboards scan-friendly

---

## Surface Priorities

### Customer

* Minimal friction
* Strong category clarity
* Fast cart interactions

### Kitchen

* Readability from distance
* Status-first design

### Waiter

* Rapid table scanning
* Immediate action visibility

### Admin

* Clear information hierarchy
* Trustworthy operational tone

---

## Anti-Patterns

* Generic SaaS cards everywhere
* Purple-on-white defaults
* Over-animated UI
* Busy visual noise without purpose
